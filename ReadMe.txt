SPECIAL NOTES:
--------------
DISPLAY FORMAT:
The game displays guessed letters in three formats:
1. Green background: Letter is correct and in the correct position.
2. Yellow background: Letter is correct but in a different position.
3. No background: Letter is not in the word.

HOW TO RUN AND COMPILE:
-----------------------
1. Ensure you have Java SDK installed on your system.
2. Navigate to the directory containing `Wordle.java`.
3. Compile the code using the following command:
>javac Wordle.java
than a class will be created
>java Wordle
